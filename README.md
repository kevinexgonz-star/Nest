K-nest
Software web de nesting 2D para optimización de corte CNC. Archivo standalone — funciona directamente en el navegador sin servidor.
✨ Funciones
 
Nesting automático de piezas irregulares con algoritmo Grid (por defecto) y motor NFP/DeepNest (opt-in)
 
Importación: SVG, DXF, PDF vectorial, Adobe Illustrator (AI)
 
Editor visual con arrastre, zoom táctil y edición de polilíneas
 
Capas de corte: Corte / Hendido / Cajeado (con offset) / Taladros / Personalizadas
 
Detección automática de capas por color al importar
 
Presupuesto con tiempo estimado y aprovechamiento por placa
 
Exportación: DXF R12 y SVG plano por capas
 
Gestión de stock con previsión de consumo y alertas
 
Persistencia en localStorage
🚀 Uso
Abrir  rotula-nest-demo-12.html  directamente en Chrome. Requiere conexión a internet en la primera carga (CDN: React, pdf.js, Babel, ClipperLib).
🛠️ Stack
React 18 · pdf.js · ClipperLib · WASM (Minkowski/offset) · Babel (JSX en navegador)
